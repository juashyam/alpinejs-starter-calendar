# ![moxie](https://github.com/juashyam/moxie/assets/13532448/9b90884e-bc58-42ca-a829-1b9ba8ce0ec0)

A basic calendar using AlpineJS and TailwindCSS.

## Installation

```
clone repository
npm install
npm run dev
```
